ssh : connexion sécurisée à distance

Connexion sécurisée avec une machine distante ssh login@ip
Connexion sécurisée avec une machine distante et affichage du bureau ssh -X login@ip
Exécution d'une commande à distance ssh login@ip bash < script
Générer une paire de clé publique/privée ssh-keygen -t rsa
Envoyer la clé publique sur l'hôte distant ssh-copy-id -i id_rsa.pub login@ip

tar : création/extraction d'une archive

Créer une archive à partir de fichiers (v : mode verbeux; f : pour spécifier les fichiers) tar -cvf nom_archive.tar fichier1 fichier2 ... fichierN
Créer une archive à partir d'un dossier tar -cvf nom_archive.tar dossier/
Créer une archive au format tar.gz à partir d'un dossier tar -czvf nom_archive.tar.gz dossier/
Créer une archive au format tar.bz2 à partir d'un dossier tar -cjvf nom_archive.tar.bz2 dossier/
Extraire une archive au format tar.gz tar -xzvf nom_archive.tar.gz
Extraire une archive au format tar.bz2 tar -xjvf nom_archive.tar.bz2
Lister le contenu d'une archive tar -tf nom_archive.tar

gzip : décompresser une archive

Décompresser un fichier .gz gzip -d fichier.gz

alias : créer un alias pour une commande

Créer un alias alias nom_alias='commande'

ln : créer un lien

Créer un lien symbolique ln -s nom_du_fichier nom_du_lien

ls : lister le contenu d'un dossier

Lister le contenu d'un dossier ls chemin_dossier
Lister le contenu d'un dossier en détail ls -l chemin_dossier
Lister le contenu d'un dossier, y compris les dossiers cachés ls -a chemin_dossier
Lister le contenu d'un dossier avec la taille des fichiers ls -h chemin_dossier
Trier les fichiers par date, du plus récent au plus ancien ls -t chemin_dossier
Trier les fichiers par taille décroissante ls -S chemin_dossier
Inverser le tri ls -r chemin_dossier
Compter le nombre de fichiers présents dans un dossier ls -al | wc -l
Compter le nombre de fichiers présents dans un dossier, sans compter "." et ".." ls -1 | wc -l

git : versionner du code

Versionner un dossier git init
Verifier l'etat d'un dossier versionné et de ses fichiers git status
Ajouter un fichier git add
Commiter un fichier git commit -m "commentaire"
Indexer et commiter un fichier a la suite (raccourci) git commit -am "commentaire"
Consulter l'historique des commit git log
Récupérer un projet sur GitHub git clone adresse_du_repository
Envoyer fichier sur GitHub (apres commit) git push origin master
Recuperer un fichier depuis GitHub git pull origin master
Verifier les branches presentes git branch
Creer une branche nommee nom_branche git branch nom_branche
Se placer sur une branche git checkout nom_branche
Fusionner une branche nommee nom_branche avec la branche actuelle (sur laquelle on se trouve) git merge nom_branche
Mettre de cote une modification (les fichiers ne prendront donc pas en compte la derniere modification en cours) git stash
Prendre en compte la modification en cours (mise de cote jusqu'a present) git stash pop

Sublime Text : l'éditeur de texte vraiment sublime !

Ouvrir un fichier cmd + P
Ouvrir un nouveau fichier cmd + O
Ouvrir la palette de commandes cmd + shift + P
Accéder aux fonctions codées dans le fichier cmd + R
Se placer à une ligne précise dans le fichier cmd + G
Sélection de plusieurs éléments simultanément cmd (maintenu) + sélection
Sélection verticale de plusieurs éléments simultanément alt (maintenu) + sélection verticale
Installer un plugin Sublim Text avec Package Control cmd + alt + P puis taper "install"
Commenter un morceau de code cmd + shift + /
